\# Customer Churn Prediction



\## Overview

This project predicts which customers are likely to churn using a machine learning classification model.



\## Objective

Identify high-risk customers early so the business can target retention actions and reduce customer loss.



\## Files

\- `generate\_churn\_data.py` — creates a synthetic customer churn dataset

\- `churn\_model.py` — trains and evaluates the churn prediction model

\- `feature\_importance.png` — top model drivers

\- `churn\_probability\_distribution.png` — distribution of predicted churn probabilities

\- `high\_risk\_customers.csv` — customers flagged as high churn risk

\- `model\_metrics.txt` — model performance summary



\## Features Used

\- Age

\- Tenure

\- Monthly spend

\- Support calls

\- Complaints

\- Digital engagement

\- Payment failures

\- Contract type

\- Product count

\- Region



\## Model

Random Forest Classifier



\## Business Value

This project helps businesses:

\- identify customers at risk of churning

\- improve retention targeting

\- reduce revenue loss

\- understand key churn drivers
